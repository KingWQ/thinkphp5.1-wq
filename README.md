thinkphp5.1-wq
===============

用thinkphp5.1框架工作中实现的一些小功能：

 1. 钉钉机器人消息通知
 2. php生成二维码
 3. redis操作类库
 4. redis+mysql做队列


#### 1: 钉钉机器人消息通知
~~~
demo/controller/DingTalk
~~~

#### 2: php生成二维码
~~~
composer require endroid/qr-code
demo/controller/Qrcode
~~~

#### 3: redis操作类库
~~~
demo/controller/Redis
~~~

#### 4: redis操作类库
~~~
common/service/QueueService
~~~
